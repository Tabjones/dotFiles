## Enabling GodMode (a.k.a. Installation)

```
git clone --recursive git@github.com:Tabjones/dotFiles.git ~/dotFiles
cd ~/dotFiles
./install
```
**Note:** Make sure you backup your old dotDirectories if they already exist, then delete them.
